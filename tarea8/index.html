Colores multinavegador

Hay más de 160 colores con combre, que son compatibles con todos los navegadores mopderno. Estos nombres se pueden usar directamente en css para aplicar colores a elementos sin especificar valores hexadecimales, rgb o hsl.

Por ejemplo :

  1. Negro
  2. Blanco
  3. Salmón
  4. Azul
  5. Lima

Colores de un borde

Para que un elemento HTML se le pueda aplicar color a su borde son necesarias 2 propiedades: border-style y border-color
