Maneras de agregar estilos a nuestra página web

Estilos en linea

Estos estilos se agregan directamente al elemento html, mediante el atributo style

Este parrafo tiene estilos en línea

Estilos internos

Estos estilos se agregan en el mismoarchivo html, dentro de la etiqueta <head> y se utiliza la etiqueta <style>

Estilos externos

Estos estilos son los que se recomiendan ya que se ofecen modularidad, facil mantenimiento, orden, eficiencia, entre otras ventejas, para agregar una hoja de estilos externa de la etiqueta <head> se va utilizar etiqueta <link>
